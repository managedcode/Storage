# Recommended Technologies

Technology choice should follow product constraints, team skill, and operating model.

## Selection Criteria

- accessibility support
- component and design-system maturity
- testability
- SSR or SPA requirements
- deployment and performance constraints

## Practical Guidance

- prefer mainstream, well-supported front-end stacks
- avoid choosing a framework only because one engineer likes it
- pick auth, styling, and state-management tools that match the product shape, not trend cycles
